In this theme I got my first tools for my toolbox.
I learned a lot of theory about different design principles such as the gestalt laws, color theory and typefaces.

01.01.02

I combined these newly attained skills in order to create a project document for the style I had to research.

The style I was given was Organic Design with focus on Saul Bass.

For the project document I created a moodboard and further elaborated choice of font, color and assets. I also had some written content in order to explain the important points about organic design.

After making my first website for mobile I continued to work towards a responsive website that would work on desktop as well.

To do so I used wireframes and created an XD prototype to visualise and test the solution.

I applied multiple gestalt laws such as common fate, figure/ground etc.

01.04.01

With the design tested at adapted I was ready to create the final project of the theme. A responsive website.

I used the wireframes and the grid property in order to make the website response according to how I drew it up.

For the mobile site I had used absolute positioning but when I set up the grid instead the whole page had a much smoother appeal and had better responsiveness.
